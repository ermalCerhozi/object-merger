## :floppy_disk: Setup

* git clone `https://github.com/ermalCerhozi/object-merger.git`.
* Install dependencies using `npm install`.
* Run `ng serve`.
* Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
